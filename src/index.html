About Tiny Pony Clipper

Tiny Pony Clipper is an open-source, lightweight background recording and remote play server designed for Linux (Wayland).

Built with Electron, WebRTC, and FFmpeg, it aims to provide a seamless experience for clipping your best gaming moments and sharing your screen with friends for low-latency remote couch co-op.

If you enjoy using this tool, consider supporting its development! You can contribute to the code, report bugs, or send a donation to help keep the project alive and improving.
